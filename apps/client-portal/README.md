# apps/client-portal

Sea King Capital — read-only Client portal + advance request submission.

## Running locally

```bash
pnpm -F @seaking/client-portal dev
```

Runs on http://localhost:3001.

## Phase 1A scope

Login page + blocking wrong-app check. The real portal (batches, invoices, remittances, advance-request submission) lands in Phase 1B+.
